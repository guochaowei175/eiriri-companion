# 英梨梨语音伴侣（Eiriri Companion）

## 📱 功能说明
- 离线语音识别
- TTS 语音合成
- 英梨梨语音包（早安、晚安、调侃语等）
- 关键词触发语音回应
- 图像头像 UI

## 🛠 构建方式（适用于 Codemagic）
1. 上传整个项目到你的 GitHub 仓库（可命名为 eiriri-companion）
2. 打开 [https://codemagic.io](https://codemagic.io)，连接你的 GitHub
3. 选择该仓库，Codemagic 会自动识别 `codemagic.yaml`
4. 点击 `Start Build`，等待生成 APK
